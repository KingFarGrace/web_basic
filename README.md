# web_basic

### 插件

+ Auto Rename Tag

  > 成对修改标签

+ Bracket Pair Colorizer 2

  > 同级括号会用相同颜色标出，便于区分层级

+ HTML CSS Support

  > 如其名

+ ESLint

  > 代码检测工具

+ IntelliSense(ES6) code Snippets

  > 快捷代码片段

+ Live Server

  > 动态在线调试

+ npm

  > 类似maven

+ npm IntelliSence

  > 自动在项目中引入npm

+ Path Intellisence

  > 自动路径补全

+ Rest Client

  > 模拟请求测试接口

+ vscode-icons

  > 提供更多图标

+ YAML

  > yaml配置文件
