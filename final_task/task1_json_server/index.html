使用json-server提供后端api实现增删改查功能

如何启动该项目？

  • 1. 在本地使用 npm 安装 json-server，控制台输入命令 npm install json-server -g；
  • 2. 进入当前目录为 task1_json_server，控制台输入命令 json-server --watch db.json 启动服务；
  • 3. 第二步完成服务启动后，即可通过api进行交互，具体使用方法见下面部分。
  • 4. 使用VSCode打开该项目所在目录，进入index.html文件，使用Alt + L + O快捷键启动前端服务器（默认使用本地服务器5500端口），若没有反应，请先在VSCode中安装live server插件后再启动，安装方法略。

关于后端api

后端使用json-server搭建虚拟服务器，并存有一批学生信息供前端进行服务请求，支持添加、修改、查询和删除四种服务，具体如下：
学生数据结构
学生假数据使用json存储，包含四个字段：
id: 学生学号；
name: 学生姓名；
age: 学生年龄；
classId: 学生所在班；
添加学生信息接口
作用描述 使用 $.ajax 将封装为json的表单数据传送给后端，并添加到后端数据集中。
url http://localhost:3000/students
请求方式 POST
{ "id": "学生学号", "name": "学生姓名", "age": "学生年龄", "classId": "学生所在班号" }
返回数据 \
修改学生信息接口
作用描述 使用 axios 将封装为json的表单数据传送给后端，并通过url中附带的id参数匹配后端数据并进行修改。
url http://localhost:3000/students/:id
请求方式 PUT
传入参数 { "id": "学生学号", "name": "学生姓名", "age": "学生年龄", "classId": "学生所在班号" }
返回数据 \
查询学生信息接口
作用描述 使用 $.ajax 将封装为json的表单数据传送给后端，并通过url中附带的id参数查询匹配项。
url http://localhost:3000/students/:id
请求方式 GET
传入参数 { "id": "学生学号" }
返回数据 { "id": "学生学号", "name": "学生姓名", "age": "学生年龄", "classId": "学生所在班号" }
删除学生信息接口
作用描述 使用 axios 将id附加到url上，并从后端数据集中删除匹配项。
url http://localhost:3000/students/:id
请求方式 DELETE
传入参数 id（非json）
返回数据 \
